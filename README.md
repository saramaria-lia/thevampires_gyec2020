# GYEC 2020 Website Submission: TransCov 
- Authors : Sara-Maria Lia (2026, 2020), Robert Stanescu (2020)

# Goal: 
This package was a part of our team's submission for the 2020 edition of the Global Youth Entrepreneurship Competition (GYEC), revisited in 2026 for fun. The name of the project is TransCov and the name of the team is TheVampires (since all 5 of the original members are from Romania). 

# History
The first version (2020) of the website can be found [here](https://vampires21.github.io), but access to the associated Github account has been lost. A more recent version (2026) of the website can be accessed [here](https://saramaria-lia.github.io/thevampires_gyec2020/).
